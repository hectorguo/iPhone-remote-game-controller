# Install
1. Fork it
2. Use `nodejs` to run `index.js`
```
node index.js
```
3. Open `index.html` on PC browser, such as http://192.168.1.10:3000
4. Open `mobile.html` on your mobile browser, such as http://192.168.1.10:3000/mobile.html

Enjoy it!
